# d3test

## Dev Tools

- `git` (Source code management)
- `node` (Runtime)
- `pnpm` (NodeJS package manager)
- `vite` (Dev server)
- `cursor` (IDE)

## Dev Technologies

- `typescript` (Language)
- `solid` (UX lib)
- `tailwindcss` (Style lib)
- `effect` (functional programming lib)
- `d3` (functional programming lib for UI)

## Getting Started

To install dependencies:

```sh
pnpm install
```

To start the dev server:

```sh
pnpm dev
```
